Polinomial

La regresin polinomial es un caso especial de regresi lineal, ya que es muy parecida a esta, sin embargo, la diferencia radica en que los datos ac no son lineales, por lo que se deben de implementar polinomios de n para obtener el modelo > Grado del polinomio Para determinar el grado del polinomio fu necesario la elaboracin de una funcin que constara con un ciclo, el cual, hara el clculo del AIC.

polyn <- function() {
for (i in c(1:10)) {
    mdl.Poly <- lm(formula = Profit ~ poly(R.D.Spend, i), data = df.Startup.Train)
    print(paste("Modelo ", i))
    aic <- AIC(mdl.Poly)
    print(aic)
}
}

SVR

SVR es uno de los principales exponentes de la aplicacin de la optimizacin de la matemtica al parendizaje supervisado, teniendo como objetivo realizar la prediccin a partir de un problema de optimizacin geomtrica con restricciones lineales.

Para determinar el mejor modelo con respecto al dataset fu necesario realizar los cuatro modelos posibles y graficarlos, para que de esta maner se supiera cul se adapta mejor a los datos.
