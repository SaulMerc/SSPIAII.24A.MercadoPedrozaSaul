Polinomial

La regresión polinomial es un caso especial de regresió lineal, ya que es muy parecida a esta, sin embargo, la diferencia radica en que los datos acá no son lineales, por lo que se deben de implementar polinomios de n para obtener el modelo > Grado del polinomio Para determinar el grado del polinomio fué necesario la elaboración de una función que constara con un ciclo, el cual, haría el cálculo del AIC.

polyn <- function() {
for (i in c(1:10)) {
    mdl.Poly <- lm(formula = Profit ~ poly(R.D.Spend, i), data = df.Startup.Train)
    print(paste("Modelo ", i))
    aic <- AIC(mdl.Poly)
    print(aic)
}
}

Y entregando el siguiente resultado, de los cuales se seleccióna los de menor valor

SVR

SVR es uno de los principales exponentes de la aplicación de la optimización de la matemática al parendizaje supervisado, teniendo como objetivo realizar la predicción a partir de un problema de optimización geométrica con restricciones lineales.

Para determinar el mejor modelo con respecto al dataset fué necesario realizar los cuatro modelos posibles y graficarlos, para que de esta maner se supiera cuál se adapta mejor a los datos.
